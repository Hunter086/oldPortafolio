HTML Introducción

Lenguaje de {etiquetas/marcas} de hipertexto

HTML (Lenguaje de Marcas de Hipertexto, del inglés HyperText Markup Language) es el componente más básico de la Web. Define el significado y la estructura del contenido web.

"Hipertexto" hace referencia a los enlaces que conectan páginas web entre sí, ya sea dentro de un único sitio web o entre sitios web.

HTML utiliza "marcas" para etiquetar texto, imágenes y otro contenido para mostrarlo en un navegador Web.

Ej: <head>, <title>, <body>, <header>, <footer>, <article>, <section>, <p>, <div>, <span>, <img>, <aside>, <audio>, <canvas>, <datalist>, <details>, <embed>, <nav>, <output>, <progress>, <video>, <ul>, <ol>, <li>

¿Qué es HTML?

  • HTML significa Lenguaje de {etiquetas/marcas} de hipertexto
  • HTML es el lenguaje de marcado estándar para crear páginas web.
  • HTML describe la estructura de una página Web.
  • HTML consta de una serie de elementos
  • Los elementos HTML le dicen al navegador cómo mostrar el contenido
  • Los elementos HTML etiquetan piezas de contenido como "este es un encabezado", "este es un párrafo", "este es un enlace", etc.

Anatomía de un documento HTML

  <!DOCTYPE html>
  <html>
    <head>
      <title>Título de la Página</title>
    </head>
    <body>
    
      <h1>Mi primer encabezado</h1>
      <p>Mi primer parafo</p>
    
    </body>
  </html>

Ejemplo

La declaración <!DOCTYPE html> define que este documento es un documento HTML5.

El elemento <html> es el elemento raíz de una página HTML.

El elemento <head> contiene metainformación sobre la página HTML.

El elemento <title> especifica un título para la página HTML (que se muestra en la barra de título del navegador o en la pestaña de la página).

El elemento <body> define el cuerpo del documento y es un contenedor de todos los contenidos visibles, como encabezados, párrafos, imágenes, hipervínculos, tablas, listas, etc.

El elemento <h1> define un encabezado grande.

El elemento <p> define un párrafo.

Elementos

Definición y uso

Todos los documentos HTML deben comenzar con una declaración del tipo <!DOCTYPE>. Esta declaración no es una etiqueta HTML. Es información para que el navegador interprete sobre que tipo de documento espera.

Para HTML 5, su declaración es:

<!DOCTYPE html>

La etiqueta <html> representa la raíz de un documento HTML. La etiqueta html es el contenedor raiz del todos los elementos HTML(sin incluir <!DOCTYPE>)

Para definir la etiqueta html es una buena practica incluir el atributo lang, para declarar el idioma de la página web.

<html lang="es"> </html>

El <head> es un contenedor de metadatos y se coloca entre la etiqueta <html> y la etiqueta <body>. Estos metadatos son datos relacionados al documento HTML y no se muestran al usuario.

Los sigientes elementos pertenecen al <head>

                        <style>
                        <title>
                        <base>
                        <meta>
                        <script>
                        <link>
                        <noscript>

El elemento <body> define el cuerpo del documento html. Este elemento contiene todo el contenido HTML(encabezados, párrafos, titulos, imagenes, links, etc).

Solo puede existir un <body> en todo el documento.

<body> </body>
